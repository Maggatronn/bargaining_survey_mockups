# Data Explorer User Guide

## Overview
The Data Explorer is an interactive tool for analyzing survey responses, exploring qualitative comments, and building insights from bargaining survey data. It helps sensemakers navigate and synthesize member feedback efficiently.

---

## Getting Started

### Login & Sensemaker Selection
1. Enter the password to access the application
2. Select your name from the dropdown to personalize your session
3. You can change your sensemaker identity later using the dropdown in the main interface

## Insights Panel (Right Side)

### Creating Insights
1. Click **+ New Insight** to start
2. Give your insight a title and add notes
3. **Cite comments** by dragging them from the Comments tab into your insight
4. Citations appear as colored tags showing the issue they relate to

### Pointers
- Create **pointers** to save and share specific views (filters, tabs, sorts)
- Click ğŸ”— **Create Pointer** on any visualization
- Pointers capture the current state so others can see exactly what you're looking at

### Managing Insights
- Edit existing insights by clicking on them
- Link insights to specific departments
- All changes save automatically to the database
